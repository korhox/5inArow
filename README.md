## Compiling and running
1. Open terminal in the project folder (`cd xxx`)
2. Install Maven depedencies:
     ```mvn install -f```
3. Compile the code with command:
      ```mvn compile -f .```
3. Run the compiled code with:
     ```java ./target/classes/fi/korho/Main.java```
